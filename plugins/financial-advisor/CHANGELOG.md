All notable changes to the Financial Advisor plugin will be documented in this file.

## [1.0.0] - 2025-02-15

### Added
- Initial release of Financial Advisor plugin
- Financial advisor agent specialized in software project financial analysis
- Cost estimation skill with 5-step framework
- Monetization planning skill with revenue model strategies
- Financial risk assessment skill with compliance focus
- ROI analysis skill with investment evaluation templates
- Quick cost estimate command for rapid estimates
- Comprehensive documentation and examples
- SaaS startup example with detailed financial analysis

### Features
- Comprehensive cost estimation for APIs, infrastructure, and development
- Revenue modeling and pricing tier design
- Financial risk identification and mitigation strategies
- ROI calculations with scenario modeling
- Support for multiple project types (SaaS, API services, mobile apps, open source)
- Compliance guidance (GDPR, HIPAA, SOC2, PCI-DSS, COPPA)
- Current pricing data (as of January 2025)

### Skills
- `cost-estimation`: Systematic 5-step cost analysis
- `monetization-planning`: Revenue strategy and pricing models
- `financial-risk-assessment`: Risk identification and mitigation
- `roi-analysis`: Return on investment calculations
