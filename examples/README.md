# Examples

This directory contains example implementations to help you understand how to create and use agents, skills, and commands.

## Contents

- [Agent Example](./example-agent/) - A sample agent configuration
- [Skill Example](./example-skill/) - A sample skill module
- [Command Example](./example-command.md) - A sample command

## Using Examples

These examples demonstrate the structure and format for contributions. Use them as templates when creating your own agents, skills, or commands.

## Structure

Each example includes:
- Clear documentation
- Usage instructions
- Sample outputs or demonstrations
- Best practices

## Getting Started

1. Review the examples
2. Copy the structure that fits your needs
3. Customize for your specific use case
4. Test thoroughly before contributing
